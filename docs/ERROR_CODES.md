## 2. Error Code 정의

### Common

- COMMON_400 : 잘못된 요청입니다.
- COMMON_401 : 인증이 필요합니다.
- COMMON_403 : 접근 권한이 없습니다.
- COMMON_404 : 요청한 데이터를 찾을 수 없습니다.
- COMMON_500 : 서버 내부 오류가 발생했습니다.

---

### Auth

- AUTH_001 : 이메일 또는 비밀번호가 올바르지 않습니다.
- AUTH_002 : 이미 가입된 이메일입니다.
- AUTH_006 : 만료되었거나 유효하지 않은 토큰입니다.

---

### Room

- ROOM_001 : 존재하지 않는 방입니다.
- ROOM_002 : 해당 방에 접근 권한이 없습니다.

---

### Scan

- SCAN_001 : 존재하지 않는 스캔입니다.
- SCAN_004 : 스캔이 아직 완료되지 않았습니다.

---

### Analysis

- ANALYSIS_001 : 분석 결과가 존재하지 않습니다.
- ANALYSIS_003 : 비교 가능한 입주/퇴거 스캔이 부족합니다.
- ANALYSIS_004 : 분석이 아직 완료되지 않았습니다.

---

### Defect

- DEFECT_001 : 존재하지 않는 하자 정보입니다.

---

### Provider

- PROVIDER_001 : 존재하지 않는 업체입니다.
- PROVIDER_002 : 추천 가능한 업체가 없습니다.

---

### Estimate

- ESTIMATE_001 : 견적 요청 생성에 실패했습니다.
